Implement the numeric market payout algorithm (D-14 through D-17). This replaces the placeholder numeric branch left in trigger_payout by Plan 05 with the full window-expansion + linear-interpolation algorithm.

Algorithm summary:
- Exact match (any prediction == resolution value): those predictors each get bet_amount * 10; others get nothing; surplus → fund
- No exact match: expand a symmetric window around resolution value until ceil(total_bettors * 0.10) fall inside
- Within window: assign ranks by distance (rank 0 = closest); multiplier = 10 - (rank / (window_size - 1)) * 9.9; each winner gets bet_amount * multiplier; surplus → fund
- Edge case: if window_size == 1, the single winner gets multiplier = 10 (full 10x)

Purpose: Completes the payout rewrite. Without this, numeric markets use the old proportional split (no cap).
Output: Numeric payout helper function + wired into trigger_payout numeric branch.

```python
def _compute_numeric_payouts(
    positions: list[tuple[uuid.UUID, float, str]],  # (user_id, bp_staked, side/prediction)
    resolution_value: float,
) -> tuple[dict[uuid.UUID, float], float]:
    """Compute BP payouts for numeric markets (D-14 to D-17).

    Args:
        positions: list of (user_id, bp_staked, prediction_str) for all active positions
        resolution_value: the resolved numeric value

    Returns:
        (payouts: dict[user_id -> bp_to_credit], total_surplus: float)
    """
    import math as _math

    if not positions:
        return {}, 0.0

    total_bp_pool = sum(bp for _, bp, _ in positions)
    total_bettors = len(positions)

    # Parse predictions as floats
    parsed: list[tuple[uuid.UUID, float, float]] = []  # (user_id, bp_staked, prediction)
    for uid, bp, side_str in positions:
        try:
            parsed.append((uid, bp, float(side_str)))
        except (ValueError, TypeError):
            pass  # skip unparseable predictions

    # D-14: exact match check
    exact_matches = [(uid, bp, pred) for uid, bp, pred in parsed if pred == resolution_value]
    if exact_matches:
        payouts: dict[uuid.UUID, float] = {}
        total_winner_payouts = 0.0
        for uid, bp, _ in exact_matches:
            payout = bp * 10
            payouts[uid] = payout
            total_winner_payouts += payout
        surplus = max(0.0, total_bp_pool - total_winner_payouts)
        return payouts, surplus

    # D-15: no exact match — expand window until ceil(total_bettors * 0.10) inside
    target_count = _math.ceil(total_bettors * 0.10)  # minimum 1
    # Sort by distance from resolution value
    sorted_by_dist = sorted(parsed, key=lambda x: abs(x[2] - resolution_value))
    # Include top target_count (or all if fewer exist)
    window = sorted_by_dist[:max(target_count, 1)]
    window_size = len(window)

    # D-16: assign multipliers by rank (rank 0 = closest = 10x)
    payouts = {}
    total_winner_payouts = 0.0
    for rank, (uid, bp, pred) in enumerate(window):
        if window_size == 1:
            multiplier = 10.0
        else:
            multiplier = 10.0 - (rank / (window_size - 1)) * 9.9
        payout = bp * multiplier
        payouts[uid] = payout
        total_winner_payouts += payout

    # D-17: surplus
    surplus = max(0.0, total_bp_pool - total_winner_payouts)
    return payouts, surplus
```
